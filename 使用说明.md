# 无限画布 · 使用说明

**版本：1.0.4**

一个 AI 多模态创作工具，三种模式自由切换：
- 🎨 **无限画布** —— 简单的 AI 文生图（适合日常用）
- 🐦 **MJ Chat** —— Midjourney 命令式聊天（Discord 风格）
- 🔗 **工作流** —— ComfyUI 风的节点式编辑器（专业设计师用）

## 1. 第一次打开

### 1.1 打开方式
- **桌面应用**：双击安装好的「无限画布」app
- **浏览器版**：双击桌面的「启动无限画布」脚本，自动打开 localhost:3000

### 1.2 必须先设置 API Key
1. 点击**左下角的齿轮图标 ⚙️**
2. 在弹出的设置框里粘贴你的 API Key（yC API 的 key，以 `sk-` 开头）
3. 点「测试」按钮验证 key 有效（绿色 ✓ 即成功）
4. 点「保存」

> Key 只存在你本机的浏览器/应用里，不会上传任何地方。

### 1.3 选模式
左侧从上到下三个图标：
- **画**（毛笔字）→ 无限画布
- **🐦**（双帆船）→ MJ Chat
- **🔗**（4 圆点 + 连线）→ 工作流

点哪个就进哪个模式。模式之间数据互通（图片共享），但展示和交互各有侧重。

## 3. 模式一：无限画布

最简单直接的 AI 生图模式。

### 3.1 基本操作

| 操作 | 效果 |
|---|---|
| 鼠标左键拖动空白 | 平移画布 |
| 滚轮 | 以鼠标位置为中心缩放 |
| 拖入图片文件 | 加到画布上 |
| 单击图片 | 选中（出现选择框 + 右上角操作条） |
| 拖动选中的图片 | 移动 |
| 拖动选中图片的角/边 | 缩放 |
| 双击空白 | 无操作（原版行为） |

### 3.2 生成 AI 图

1. 在右侧面板的输入框里**写描述**（中英文都可）
2. **选模型**（下拉框）—— GPT Image / Flux / Seedream / Gemini / Grok / Qwen / MJ / Ideogram 等
3. **选比例**（1:1 / 16:9 / 9:16 / 4:3 / 3:4 / 3:2 / 2:3）—— 不同模型支持的比例不同，会自动锁定
4. **选清晰度**（1K / 2K / 4K）—— 同上，按模型自动锁定
5. **选数量**（1 / 2 / 4 张）
6. 点「生成」按钮 或 按 **Cmd/Ctrl + Enter**

占位框立刻出现，AI 完成后图片自动到位。

> 多张同时生成时，占位框排成 1×1 / 1×2 / 2×2 网格。

### 3.3 参考图（垫图）

让 AI 在你提供的图基础上修改/参考：

**方法 1：从画布加**
- 单击一张已有图 → 它的右上角弹出小工具条 → 点 ⊞ 图标 → 加为参考

**方法 2：从本地上传**
- 点右侧面板「+」按钮 → 选文件

**方法 3：拖文件**
- 直接把图片文件拖到右侧"参考图"区域

最多 8 张参考图。点参考图缩略图右上角的「×」可移除单张，或点「清除全部」清空。

### 3.4 在 prompt 里 @ 引用参考图

在描述输入框里输入 `@`，弹出参考图缩略图列表：
- 上下方向键选择
- Enter / Tab 确认
- 输入的位置会变成 `@1`、`@2` 之类的标记 —— 告诉 AI 在哪里使用某张参考图

### 3.5 已有图的右键菜单

右键画布上任意一张图（不是占位框）：

| 菜单项 | 作用 |
|---|---|
| 设为参考图 | 加入参考图列表 |
| 画笔标注 | 进入涂抹模式（圈出区域 + 写说明 → 当作改图指令） |
| 高清放大 (2K) | 用 Gemini 把这张图放大到 2K |
| 下载图片 | 保存到本地 |
| 删除 | 从画布删 |

### 3.6 选中图片的悬浮工具条

选中一张图时，右上角会出现 3 个小按钮（快捷版右键菜单）：
- ⊞ 设为参考图
- ✏️ 画笔标注
- ↓ 下载

不想看到？点空白处取消选中。

### 3.7 模型支持的比例和分辨率

不同模型支持的参数不同，选了模型后比例和清晰度下拉会**自动锁定**到该模型支持的范围：

| 模型 | 支持比例 | 支持分辨率 |
|---|---|---|
| GPT Image 1/1.5/2 | 全部 7 种（实际 snap 到 1:1 / 3:2 / 2:3） | 1K |
| Sora Image | 1:1 / 16:9 / 9:16 | 1K |
| Gemini 2.5 Flash Image | 全部 7 种 | 1K |
| Gemini 3 Pro Preview | 全部 7 种 | 1K / 2K |
| Flux Schnell / Dev | 全部 7 种 | 1K |
| Flux Pro 系列 | 全部 7 种 | 1K / 2K |
| Seedream 3.0 | 全部 7 种 | 1K / 2K |
| Seedream 4.0+ | 全部 7 种 | 1K / 2K / **4K** |
| Grok 3/4/4.1 | 5 种 | 1K |
| Grok Imagine | 全部 7 种 | 1K / 2K |
| Qwen Image Max | 全部 7 种 | 1K / 2K |
| MJ Imagine | 全部 7 种 | 1K |
| Ideogram V1/V2 | 全部 7 种 | 1K |
| Ideogram V3 | 全部 7 种 | 1K / 2K |

## 4. 模式二：MJ Chat

Discord 风格的 Midjourney 命令式聊天。

### 4.1 基本输入

输入框写 `/` 会弹出命令菜单：

| 命令 | 作用 |
|---|---|
| `/imagine <prompt>` | 生成 4 宫格 |
| `/describe` | 给一张图，AI 反推 4 个 prompt 候选 |
| `/blend` | 混合 2-5 张图 |
| `/relax` | 切到 Relax 队列（慢但便宜） |
| `/fast` | 切到 Fast 队列 |
| `/turbo` | 切到 Turbo 队列（最快） |
| `/info` | 查询账号信息 |
| `/subscribe` | 订阅链接 |
| `/help` | 帮助 |
| ... | 总共 25 条官方命令 |

### 4.2 提交方式

- **Enter**：发送
- **Shift+Enter**：换行
- **/imagine** 后面跟 prompt，可加参数：
  - `--ar 16:9` → 横版
  - `--niji 6` → 切到 Niji（动漫风）
  - `--v 7` → 切到 v7 模型
  - `--style raw` / `--stylize 200` 等

### 4.3 垫图

输入框上方有"参考图"小条，最多 5 张：
- 拖文件进来
- 复制图片后 Cmd/Ctrl + V 粘贴
- 点 + 按钮选文件

### 4.4 任务结果

每个生成的 4 宫格出现后，下面有 8 个按钮：
- **U1/U2/U3/U4** —— 选哪一张做高清放大
- **V1/V2/V3/V4** —— 选哪一张做变体
- **🔄** —— 重新生成
- **Vary Region** —— 局部变体（涂抹一块区域 + 写改图描述）

### 4.5 任务恢复

如果你**刷新页面**或**关掉再打开**，进行中的 MJ 任务会显示一个「恢复」按钮 —— 点了就继续等待结果。任务完成后聊天记录里会自动出现。

### 4.6 把 MJ 图带到画布

任意 MJ 结果图：
- **双击**：把这张图加到无限画布主区
- **右键 → 设为参考图**：加为画布的参考图（不切模式）

## 5. 模式三：工作流

ComfyUI 风的**节点式编辑器**，适合做多步骤创作流程的设计师。

### 5.1 一开始：完全空白

切到工作流模式，画布是干净的（没有右侧栏，没有底部条）。所有内容都靠你自己添加。

### 5.2 添加节点 —— 双击空白

双击画布空白处，弹出菜单：

```
┌──────────────────────────────────┐
│ 🖼️ 图像生成   AI 文生图 / 改图     │
│ 💬 语言大模型 文本生成 / 改写       │
│ ─────────                       │
│ 📝 文本节点   便签 / 标注          │
│ ↑ 上传图片   从本地选择           │
└──────────────────────────────────┘
```

### 5.3 图像生成节点

点「图像生成」→ 一张**带表单的卡片**出现在双击位置：

```
┌───────────────────────────────┐
│ 🖼️ 图像生成        ⌛/✗  ×    │ ← 头部（拖动 / 状态 / 删除）
├───────────────────────────────┤
│ ┌───────────────────────────┐ │
│ │ 描述你想生成的图片...        │ │ ← prompt 输入
│ │                           │ │
│ └───────────────────────────┘ │
│ [模型 ▾]                      │
│ [比例 ▾]      [清晰度 ▾]       │
│             [    生成    ]    │
└───────────────────────────────┘
```

**操作**：
- 写 prompt（textarea 自动聚焦）
- 选模型 / 比例 / 清晰度
- 点「生成」（或 Cmd/Ctrl + Enter）
- 等待几秒到几十秒
- **节点本身**会变成生成出来的图片（卡片消失，图片占据原位）

**移动**：按住头部的"🖼️ 图像生成"那一行往任意方向拖。  
**删除**：点头部右上角的 ×。

### 5.4 语言大模型节点

点「语言大模型」→ 一张**简化版的表单卡片**出现：

```
┌───────────────────────────────┐
│ 💬 语言大模型      ⌛/✗  ×    │
├───────────────────────────────┤
│ ┌───────────────────────────┐ │
│ │ 给 AI 一段指令…             │ │
│ └───────────────────────────┘ │
│ [模型 ▾]                      │
│             [    生成    ]    │
└───────────────────────────────┘
```

**支持模型**：GPT-4o / GPT-4o mini / GPT-4.1 / GPT-5 / Claude Haiku/Sonnet/Opus 4.5 / Gemini 2.5 Pro/Flash / DeepSeek Chat/Reasoner。

**用法举例**：
- 把一段中文翻译成英文
- 给我扩展一个画图 prompt（"中世纪城堡在月光下" → 详细描述）
- 写一段产品介绍文案
- 把一段长文本概括成 3 句话

生成完成后，**节点本身**变成一张暖黄色便签（文本节点），里面是 AI 返回的文字。

### 5.5 文本节点

点「文本节点」→ 一张**暖黄色便签**出现，并自动进入编辑模式。

**编辑**：直接打字。
- **Cmd/Ctrl + Enter**：保存退出编辑
- **ESC**：取消（不保存）
- 点便签外任何地方：自动保存

**之后**：双击便签可再次编辑。  
**右键便签**：
- 编辑文本
- **用此文本作为 prompt** → 把内容塞到下一次生成

### 5.6 上传图片

点「上传图片」→ 选本地文件 → 图片落到双击位置。

### 5.7 工作流的灵魂：自动连线

当某个节点是"基于其他节点"生成的，画布会自动**画一条暖橙色弧线**连接它们。

**最常用的方式**：
1. 选中画布上一张图
2. 右键 → **「✨ 基于此图再生成」**
3. 一个**新的图像生成节点**出现在源图右侧，源图已自动填入为输入
4. 写新 prompt → 点生成
5. 完成后：**新图自动连线回源图**，形成"血缘"

这就是 ComfyUI 风的核心 —— 你能直观看到一棵创作树，每张图来自哪里、又派生出什么。

### 5.8 工作流导出 / 导入（备份和分享）

**左下角的 ⊞ 工作流按钮**（仅工作流模式可见）：

| 选项 | 作用 |
|---|---|
| ↓ 导出工作流 | 把当前画布的所有节点 + 连线打包成 JSON 文件下载 |
| ↑ 导入工作流 | 选 JSON 文件，**合并**到当前画布 |
| ⟲ 导入并替换 | 先清空当前画布，然后导入 |

> 导出的 JSON 包含图片的 base64，所以文件会比较大（几 MB 到几十 MB）。可以用来：
> - 备份重要工作流
> - 发给朋友复刻
> - 在不同设备间迁移

### 5.9 工作流模式的右键菜单

**右键图片**：
- ✨ **基于此图再生成**（仅工作流模式显示）
- 设为参考图（暂时画布模式专用）
- 画笔标注
- 高清放大 (2K)
- 下载图片
- 删除

**右键文本节点**：
- 编辑文本
- 用此文本作为 prompt
- 删除

**右键画布空白**：无菜单（请用双击）

### 5.10 工作流和画布模式的关系

数据是共用的，但显示不同：
- 你在工作流模式建的**文本节点 / 表单节点 / 连线** → **切回画布模式后看不见**（被过滤）
- 你在画布模式生成的图 → **切到工作流模式照样能看到**
- 切回工作流模式，之前建的所有内容**完整恢复**

## 6. 通用功能

### 6.1 设置（左下 ⚙️）

- **API Key**：必填，yC API 的 key
- **测试**：发请求验证 key + base 是否有效
- **保存**：写入 localStorage
- **清空**：清掉本地保存的 key

主题切换（深色 / 浅色 / 自动跟随系统）也在这里。

### 6.2 诊断面板（左下 💓）

点心电图图标 → 右侧滑出面板，显示**最近 200 条 API 请求**：
- ⏱️ 耗时
- ✓/✗ 状态
- URL / 请求体（点击展开看完整 JSON）
- 响应内容（最多 4KB）
- 复制按钮（请求 / 响应 / 完整记录）

**用途**：生成失败时，看看是什么报错；想了解后端到底发了什么数据。

红点 ● 表示有失败的请求 —— 看一眼就消失。

### 6.3 清空所有（左下 🗑️）

把当前画布的所有节点 + 连线**全部删除**。会先弹确认框告诉你删多少个。

> 不影响视图位置 / 缩放，只删内容。
> 删了**没法撤销**！重要工作流先导出 JSON 备份。

### 6.4 工作流（左下 ⊞）

仅工作流模式可见。功能见 5.8。

### 6.5 持久化

所有内容自动存 IndexedDB（不是 localStorage）：
- 画布上的图片（base64）
- 节点位置 / 大小
- 视图缩放和平移
- 工作流连线
- 文本节点内容
- 表单节点的草稿（写一半 prompt 刷新页面，回来还在）
- MJ 聊天记录
- 诊断日志

刷新页面、重启应用、断网都不会丢。

### 6.6 暗色模式

设置里有 3 个选项：
- 浅色（暖米黄）
- 深色（Discord 蓝灰）
- 跟随系统

工作流里的连线、节点头部条等所有元素都做了双主题适配。

## 8. 常见问题

### Q: 生成失败显示"上游错误"
A: 步骤：
1. 检查 API Key（左下齿轮 → 测试）
2. 打开诊断面板（左下 💓）→ 看最近的红色失败记录 → 复制响应文本，看具体报错
3. 检查模型是否还在服务（yC 可能下架某些模型）

### Q: GPT Image 选了 4K 但实际输出尺寸很小
A: GPT Image 系列原生只支持 1024² / 1536×1024 / 1024×1536 三种尺寸。后端会把任意比例 snap 到这 3 种。**想要真正 4K 输出，用 Seedream 4.0+。**

### Q: MJ 的 /describe 报错
A: `/describe` 不支持 Niji 模型。如果你切到了 Niji，先 `/imagine` 切回 MJ Bot 再试。

### Q: 工作流的连线看不到
A: 几个可能：
- 缩得太小（zoom < 30%），连线很细看不清
- 该图没有 sources 字段（不是从其他节点派生的，是手动上传或独立生成的）
- 在画布模式下查看（连线只在工作流模式显示）

### Q: 我的图怎么都没了
A: 可能性：
- 浏览器清了 IndexedDB（隐私模式 / 手动清缓存）
- 切到了不同的浏览器 / 不同的电脑
- 应用版本升级出了 bug

**预防**：定期用工作流的「导出 JSON」功能备份重要内容。

### Q: 文本节点的内容能搜索吗？
A: 暂不支持。你可以 Ctrl+F 搜索浏览器页面上可见的文字（zoom 大到能看清的话）。

### Q: 文件传到 Windows 朋友那里他装不上
A: **不要用微信传 exe / dmg 文件**！微信会自动压缩破坏二进制。用 U 盘 / 网盘 / WeTransfer / 邮件附件。

### Q: Mac 上提示"已损坏，无法打开"
A: 这是 Gatekeeper 的 quarantine 标记。在终端跑：
```
sudo xattr -rd com.apple.quarantine /Applications/无限画布.app
```

### Q: 工作流的图像生成节点能改大小吗？
A: 暂时不能（卡片是固定 320×280 世界单位）。但可以**拖动头部把它搬到任何位置**。下个版本会加节点 resize 手柄。

### Q: 能在工作流里手动拉线连接两个节点吗？
A: 暂时不支持手动拉线。当前的连线**全部是自动**生成的：右键"基于此图再生成"会自动建立连线。手动拉线（拖拽连接点）是下一个迭代要做的功能。

### Q: 怎么从工作流清掉所有节点重新开始？
A: 左下角的「🗑️ 清空所有」按钮。会先弹确认框告诉你要删多少个。

### Q: 生成的图能编辑吗？
A: 右键 → **画笔标注**（仅画布模式）→ 在图上涂抹一块区域 + 写改图描述 → 触发"局部编辑"重新生成。
